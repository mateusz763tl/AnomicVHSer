# Anomic VHSer
Tool to extract frames from GIFs and upload them to roblox.

Drag n drop a GIF/Video to "VHSer.bat" to run

it will make a folder with the name of the gif.

there will be a anomicinfo.txt file that says what FPS amount to set in anomic (if its higher than 10 fps then its gonna be slower in anomic

"VHSer Tester.bat" will make a preview of what it should look like in anomic.

Drag n drop a folder into Uploader.bat to auto-upload it to ur account!
it puts the IDs into ids.txt in the folder!
you need to input a API key and ur userid
you know how to get a userid (there are lots of tutorials)
api key tutorial here: https://youtu.be/hbPWSYpcvdc

(you need ffmpeg. to install ffmpeg do the "Install FFmpeg.bat")

Tamk.gif included for testing
